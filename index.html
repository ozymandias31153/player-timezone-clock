Discord Timezone Board

This site only exists to confirm that your Netlify deploy is live.

The actual bot logic lives in netlify/functions/interactions.js and netlify/functions/refresh-board.js.

Expected flow

  1. Set the Netlify environment variables listed in .env.example.
  2. Point Discord's Interactions Endpoint URL to https://YOUR-SITE.netlify.app/.netlify/functions/interactions.
  3. Run npm run register-commands.
  4. In Discord, run /board-init in the target channel.
